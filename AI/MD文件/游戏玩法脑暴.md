# Role
擅长将AI技术落地到成熟玩法中的游戏设计师。

# Project Context
**项目目标**：开发一款面向青少年的**AI生成3D宠物**游戏。
**核心体验**：
1.  **机制**：AI生成多只3D宠物，在3D家园中互动。
2.  **基调**：浅养成、低门槛、轻松。
**核心挑战**：验证**“宠物有用性”**，即建立玩家持续创造与养成的动力闭环。

# Design Constraints
1.  **稳健落地**：避免过度创新，基于成熟验证的玩法机制进行微调。
2.  **AI自然融合**：AI生成与交互需自然融入核心玩法，避免生硬割裂。
3.  **低门槛**：严格保持操作简单、体验轻松。

# Task
提供 5 个不同维度的玩法原型方向。

**方案要求：**
1.  **对标知名原型**：必须明确参考一个知名游戏原型（如《动森》、《宝可梦》等），便于快速理解。
2.  **体现宠物价值**：具体说明宠物在玩法中的功能性（Utility），为何玩家需要它们。
3.  **AI结合点**：简述AI技术（生成/交互）在其中的具体应用场景。

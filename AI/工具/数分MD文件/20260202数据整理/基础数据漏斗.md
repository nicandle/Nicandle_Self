# 孵化领取率需求描述

## 1. 基础信息
- **SQL文件名称**: 前期漏斗
- **数据库**: `nvwa_cbt1`
- **引擎要求**: Spark SQL

## 2. 核心需求
- 统计新手引导日志中每一步玩家的完成数
- 统计任务日志中每条任务的玩家完成数
- 按照给定的顺序和结构混合排列这些引导和任务


## 3. 字段信息
- 字段lock_region，参数为 ID 或者 BR
- 日志AccountRegister，local_dt字段注册日期为20260129至20260130
- GuideStep日志中Guide_id为引导id，若完成则会上报。没有上报则未完成
- TaskAction日志中存在某条task_id且task_action字段值为2时，视为任务完成；否则无


## 4. 期望输出字段
输出格式和顺序参考文件 Nvwa_gr2_数据 - 前期漏斗0129.csv

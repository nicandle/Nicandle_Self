# 批量配置与看图工具使用说明

## 1. 快速开始
1. 确保电脑已安装 Python 环境。
2. 双击运行文件夹内的 **`一键启动.bat`**。
   - 该脚本会自动安装必要的依赖库 (`pandas`, `openpyxl`, `pillow`)。
   - 安装完成后会自动打开工具界面。

## 2. 功能一：生成配置
此功能用于将生物、修饰词、实体词进行全排列组合，并导出 Excel。

### 输入格式
请在界面的三个输入框中分别输入数据。
- **格式要求**：每行一条数据，包含 **ID** 和 **名称**，中间用空格或制表符分开。
- **数据来源**：可以直接从 Excel 的两列复制粘贴。

**示例数据：**
**第一列：生物**
```text
1001 狮子
1002 老虎
```

**第二列：修饰词**
```text
201 火焰
202 冰霜
```

**第三列：实体词**
```text
301 雕像
302 玩偶
```

### 导出结果
点击 **“生成并导出Excel”**，将生成包含 5 列的表格：
1. **流水ID**：唯一数字 ID（1000001 起始）。
2. **组合名称**：如 `狮子 火焰 雕像`。
3. **生物ID**：如 `1001`。
4. **组合ID**：格式为 `修饰词ID|实体词ID`，如 `201|301`。
5. **资源名称**：系统自动生成的唯一标识，格式为 `Res_生物ID_修饰词ID_实体词ID`（如 `Res_1001_201_301`），用于图片匹配。

---

## 3. 功能二：图片查看与筛选
此功能用于查看资源图片，并筛选出质量不符的图片导出。

### 操作步骤
1. **加载配置**：
   - 点击 **“1. 加载配置Excel”**。
   - 选择在上一步生成的 Excel 文件。
   - *注意：这一步是必须的，工具需要读取 Excel 中的“资源名称”与“组合名称”的对应关系。*

2. **加载图片**：
   - 点击 **“2. 选择图片文件夹”**。
   - 选择存放图片的目录。
   - **匹配规则**：图片的文件名（不含扩展名）必须与 Excel 中的 **资源名称** 一致（例如图片名为 `Res_1001_201_301.png`）。
   - 加载成功后，图片下方会显示对应的中文组合名称。

3. **筛选与导出**：
   - **筛选**：在列表中浏览图片，点击图片或文字即可 **选中**（选中项背景变红），表示该图片质量不符或需要标记。
   - **导出**：点击 **“3. 导出选中(不合格)列表”**，将生成一个新的 Excel 文件，包含所有被选中的图片文件名及其对应的组合名称。
